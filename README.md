🚀 DataLens — AI-Powered Data Analysis Engine

DataLens is a browser-based AI data analysis tool that transforms raw CSV files into clean, structured datasets with rich insights, visualizations, and automated reports — all in seconds.

✨ Features
📂 1. CSV Upload
Drag & drop or file picker
Supports datasets up to 50MB
Instant parsing using PapaParse
🧹 2. Automated Data Cleaning
Trims whitespace
Standardizes null values (NULL, N/A, -, etc.)
Removes:
Empty rows
Duplicate rows
Fills missing numeric values using median imputation
Cleans currency and percentage symbols
Normalizes text fields (names, cities, etc.)
📊 3. Data Preview
Displays first 50 rows of cleaned dataset
Highlights:
Numeric values
Dates
Missing values
📈 4. Statistical Analysis

For each column:

Numeric Columns
Min, Max
Mean, Median
Standard Deviation
Categorical Columns
Unique values
Most frequent value
Missing data percentage visualization
📉 5. Visualizations

Powered by Chart.js

Histograms (numeric distribution)
Bar charts (categorical frequency)
Scatter plots (relationships between variables)
🤖 6. AI Insights
Generates structured analysis using LLM
Includes:
Overview
Key findings
Data quality issues
Patterns & correlations
Actionable recommendations
📥 7. Export Options
Download cleaned dataset (.csv)
Download full analysis report (.txt)
